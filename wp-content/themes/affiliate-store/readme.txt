== Description ==

The Affiliate Store WordPress Theme is a powerful and versatile solution designed specifically for building high-converting affiliate marketing websites. Ideal for beginners and seasoned marketers alike, this theme provides a user-friendly platform to create affiliate stores that showcase affiliate products and drive conversions. With its customizable layout, the theme allows users to implement unique affiliate store branding, helping to establish a distinct online presence. The theme includes various features such as affiliate product listings, affiliate landing pages, and affiliate link integration, making it easy to promote affiliate products for blogs and other platforms. Additionally, the visual elements are designed to attract visitors, with sleek product pages, engaging banners, and compelling calls to action that encourage affiliate promotions. Benefits include streamlined affiliate tracking and management through affiliate store plugins for WordPress, ensuring that users can monitor their performance and optimise their strategies effectively. The Affiliate Store WordPress Theme also supports affiliate product SEO, enhancing visibility in search engines and attracting more organic traffic. With tools for affiliate content creation and management, users can develop effective affiliate marketing campaigns tailored to their niche. Whether you are focused on affiliate marketing for eCommerce or looking to create an affiliate referral program, this theme provides everything needed to succeed in the competitive affiliate marketing landscape.

== Homepage Installation ==

Step-1. Go to Dashboard -- Pages -- Add new page -- Add templates (Home Page) -- Publish it.
Step-2. Go to Dashboard -- Settings -- Reading -- Click on static page -- Select home page -- Save changes.

Topbar Section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Topbar Section -- Check To Enable This Section -- Add Text -- Check to Show Curreny Switcher -- Middle Header Image -- Add Text -- Add Links -- Publish it.

Slider Section
Step-1. Go to Dashboard -- Posts -- Add new Post -- Add title, content, and featured image-- Publish it.
Step-2. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Slider Section -- Select Category to display Latest Post -- Add Slider Subheading -- Add Categories Title -- Select About Image -- Add Offer Title -- Add Button Text -- Add Button URL -- Publish it.

Our Team Section
Step-1. Go to Dashboard -- Products -- Add new Products -- Add title, content, and featured image-- Publish it.
Step-2. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Trending Products Section
 -- Fill the our team information. -- Publish it.

Footer Section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Footer Section -- Add copyright text -- Publish it.
